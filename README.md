## Tools

- Color name (API by [meodai/color-name-api](https://github.com/meodai/color-name-api))
- Color shade generator
- Semantic color generator
- SASS Complier

## Local Development

Install dependencies:

```sh
yarn
```

Starting local server:

```sh
yarn dev
```
